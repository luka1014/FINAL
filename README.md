рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃџрЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃърЃарЃўрЃюрЃфрЃўрЃърЃћрЃЉрЃўрЃА рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃљрЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃБрЃцрЃЌрЃљ, рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў HTML/CSS рЃЎрЃЮрЃЊрЃўрЃА рЃЊрЃљрЃгрЃћрЃарЃљ.
